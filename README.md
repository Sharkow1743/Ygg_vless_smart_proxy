# Установка
Запустите `startup.bat` и следуйте инструкциям

# Что скрипт делает
 - Устанавливает mesh-сеть `Yggdrasil`
 - Настраивает её чтобы она могла работать
 - Запускает прокси, который маршрутизирует трафик через cloudflare warp


# Информация
После установки прокси будет запускатся автоматически при запуске `Windows`

Вы можете запустить прокси вручную, используя `Netch.exe`

Получить доступ к настройкам прокси можно через иконку в трее
